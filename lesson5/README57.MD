## 🎨 **介面特色**

### **1. 美觀設計**
- 使用漸層背景的標題區域
- 現代化的Soft主題
- 響應式佈局，最大寬度1200px
- 圓角邊框和陰影效果

### **2. 功能完整**
- **多語言支援**：英文、日文、韓文、法文、德文、西班牙文等
- **專業領域選擇**：商業、科技、醫學、法律、教育、文學等
- **長文本支援**：可處理多行文本翻譯
- **錯誤處理**：包含完整的異常處理機制

### **3. 用戶體驗**
- **直觀操作**：左右分欄設計，輸入和輸出清晰分離
- **實用按鈕**：清除、複製結果功能
- **使用範例**：內建範例幫助用戶快速上手
- **即時反饋**：翻譯按鈕有明確的視覺提示

### **4. 技術特點**
- 使用LangChain的ChatPromptTemplate
- 整合Ollama本地模型
- 支援自定義CSS樣式
- 完整的錯誤處理和用戶提示

## 🚀 **如何使用**

   ```bash
   python lesson5_7.py
   ```

2. **開啟瀏覽器**：
   - 訪問 `http://localhost:7860`
